Um ecossistema React & Node projetado para achar a mina de ouro invisível do Google Maps: Empresas que faturam físico, mas não possuem um site na internet.

React 18 Supabase Node + Puppeteer

1. O Radar (Data Mining)

Acabou o Ctrl+C e Ctrl+V no Google Maps. O Front-end bate em uma Edge Function segura que varre cidades inteiras e cospe milhares de leads de uma vez.

O Pulo do Gato (O Código):

A lista negra (Blacklist) roda antes de você ver a tela. Advogados da união e concorrentes são destruídos do Array. Só sobra lead qualificado.

2. Encontrando quem "Sangra" (A Falta de Site)

O algoritmo varre centenas de alvos na hora. Se o Google responde que o campo website está vazio... Bingo! O sistema coloca uma tarja vermelha absurda na sua tela informando "SEM SITE". Essa é a empresa que está perdendo dinheiro.

Extra - Auditoria Profunda:

Se a empresa TEM um site, não damos trégua. Batemos na API do PageSpeed. Se faltar certificado SSL ou estiver lento no celular, a gente pinta ele de vermelho também!

3. A Fábrica de Conversão (WA Standalone)

O painel joga o lead pra fila do Supabase. Daqui pra frente você não faz nada. Um Robô em Node (totalmente isolado da web pra evitar Bans da Meta) lê o banco. Ele pede para a Inteligência Artificial (Groq) montar uma mensagem hiper-pessoal sobre a dor do cliente.

A Mágica do Puppeteer (PDF On-The-Fly):

Fazer o navegador do usuário gerar PDF toda hora travaria ele. Nós levantamos uma porta :3001 escondida num servidor Node. Um Chrome fantasma tira "foto" das cores do cliente e envia um PDF VIP via API direto no número de zap dele como Base64!
